## Getting Started

### Prerequisites

- Node.js 18+

### Setup

```bash
git clone https://github.com/ENZOMOTIVE/Farm-formosa-App.git
cd Farm-formosa-App
```

```bash
cd farm-formosa
npm install
npm start
```
